# matrixComputation
A repository that implements algorithms in the book "Matrix Computations", Fourth Edition, by Gene H. Golub and Charles F. Van Loan
## Structure : 
- [`matrixDecomp.py`](matrixDecomp.py)
    - LU Decomposition
    - PLU Decomposition
    - QR Decomposition
    - Bidiagonalization
    - Tridiagonalization
